REPORT


In this project, I created a Student Portal webpage using HTML and CSS. The page includes a navigation bar, a student information table, and a form to collect student details.

The navigation bar helps users quickly access different sections like the table, forms, and report. The table displays student records in a structured way, making it easy to read. The form allows users to enter new student information, which can be submitted for processing.

The purpose of this project is to provide an organized way to manage student data. It is simple, user-friendly, and easy to navigate.
