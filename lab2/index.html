Tags and their purposes :

  • <div>: Used as a container element.
  • <section>: Represents different sections of content on the page.
  • <h>: Header tags used to define headings.
  • <p>: Paragraph tag used to define blocks of text in the content.
  • <img>: Image tag used to insert an image.
  • <nav>: Defines the navigation section.
  • <ul>: Creates an unordered list.
  • <li>: Defines individual list items within a list.
  • <a>: Anchor tag used to define hyperlinks.
